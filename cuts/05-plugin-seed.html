The first thing Liminal does is not ask for a brief. It reads. Each bounded agent reads its own source, in parallel, and posts the one thing worth acting on — so the first deliberation starts warm, not from a blank table.

03 · /try-liminal · the loop in 60 seconds
$ /try-liminal "is the ~240-finding reclassification a triage-quality problem?"

Three agents read the same brief. The one in lane does the work; the others refuse and name the right agent. The refusal is the routing — not an error.

Analystin lane · Diligence
The reclassification is not a quality failure — it is the absence of a shared P-tier lexicon meeting a novel frontier-model output with no calibration phase. ~240 P1↔P2 moves across two passes is what happens when "moderate" has no agreed definition to drift from. The pattern is systemic, not individual: the same reviewers are inconsistent in the same direction, which reads as a missing rubric, not a missing skill.
